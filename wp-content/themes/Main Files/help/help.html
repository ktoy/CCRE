1) About The Theme - top

This theme was built and designed to work with version 3.0. Please upgrade your Wordpress to version 3.0. if you have not already. The theme has not been tested with previous versions of WordPress.

The theme is widget ready and you can add and configure your own widgets in the side bar however you like. There are several color schemes installed with the theme. You can activate any of these from the theme options menu.

2) Installation - top - top

Installing the theme can be done two ways. You can upload the zipped theme file, using the built in WordPress theme upload, or copy the files through FTP.

A) Using WordPress Theme Upload

  1. Login to your WordPress admin.
  2. In the Appearance menu click, "Themes"
  3. At the top of the page click, "Install Themes" tab then click "Browse"
  4. Select the zipped theme file and click "Install Now"
  5. After install you will receive the message, "Theme Installed successfully."
  6. Click the link "Activate"

B) Uploading by FTP

  1. Login to your FTP server and navigate to your WordPress themes directory. Normally this would be "wp-content/themes"
  2. Extract the files from the zipped theme.
  3. Copy the folder "LOP" to your themes directory.
  4. After the files copy, login to your WordPress admin.
  5. In the Appearance menu click "Themes"
  6. Click "Activate" for the theme "LightofPeace"

Once the theme is installed you are ready to start configuring it for your content. See the section on configuring the theme options below.

3) Configuring Theme - top

You need to perform these steps, otherwise the theme will not display properly

Home Page setup

You need to change the default homepage to the theme home page template.

  1. Login to your WordPress admin.
  2. In the Pages menu click, "Add New". Create a new page called "Home"
  3. In the Page Attributes menu, choose Site_Home in the Template drop down."
  4. In the Settings menu click, "Reading"
  5. Choose "Home" for the Front page in the Front page displays.
  6. Save Changes

Menu Setup

  1. Create a custom menu, for example "Menu 1"
  2. After creating the custom menu, you have several options on Menu Management page to add links. For example, you can simply check the boxes next to the Pages and Categories you want to add then click the Add to Menu button.
  3. Create 2 custom menus, one for the Primary Menu and one for Secondary Menu
  4. Select the location for each menu in the Theme Location window. For example "Menu 1" for Primary Menu and "Menu 2" for Secondary Menu.

Random Page Header Image Setup

Each page template will show a header image randomly. All the header images is in the img > headers folder. You can specify a certain image for a certain page. If you not specify an image, it will show a random image.

You need to edit the page id in header-images.php to do this.

Example:

  1. Open any page in the Pages menu
  2. Find the page id
  3. Fill the page id in the header-images.php with the page id you found in the Wordpress admin

    For example, the page with ID 2 will show header-1.jpg. You can add as many header image as you want.

Admin Panel

Once the theme is installed you will see a new menu on the bottom left side of your WordPress admin. The theme menu contains the following areas:

  • General Options

    This area contains settings for choosing the color scheme and changing your logo.

  • Home Page

    The home page section is where you will select the content source for the 3 tabs on the home page. It also contains options for configuring your slide show.

    You should select the category you will be using for your tabs in this area so you don't forget. If you don't have a category you want to use, create one and select it now.

  • Sidebar

    This is where you fill the content for the Headline Box

  • Footer

    This is where you fill the content for the right column footer. You also can change your footer logo and add tracking code.

Popup Window Setup

This is a little bit complicated. You need to edit popup.js. Follow these steps and you should be good.

  1. Create a custom link, for this example I created "Service Times" link.
  2. Leave the URL blank.
  3. Preview you site in the browser. Right click and choose View Source.
  4. Find the "Service Times" ID and copy it.
  5. Open popup.js, it's located in the js folder. And paste the ID in line 62.
  6. Edit popup.php and fill the content for the Popup window.

4) HTML Structure - top

This theme is a fixed layout with two columns. All of the information within the main content area is nested within a div with an id of content-left. The sidebar's content is within a div with an id of content-right. Here is the general structure of the typical page:

  • #wrapper div holds the header and content.
  • #main div is used to center its content (#content and #header).
  • #header div contains logo, top menu, and navigation bar.
  • #content div is the main content area of the site. It holds #content-left and #content-right (most people called it sidebar).
  • #popupContact div holds the popup window for service times and direction.
  • #footer consists of secondary menu items and contact information.
